# Web Scraping: Largest Companies in India

## Project Overview
This project scrapes data about the largest companies in India from Wikipedia
using Python and BeautifulSoup.

## Tools Used
- Python
- Requests
- BeautifulSoup
- Pandas

## Output
- CSV file containing company details

## How to Run
1. Install dependencies:
   pip install requests beautifulsoup4 pandas
2. Run the notebook:
   jupyter notebook Data_Scraping.ipynb
